Purpose-Built

Everything the Web UI Does — Faster

Native Performance
A Go core via Wails. Opens instantly, sips memory, and stays responsive over millions of events. No Electron bloat.
SQL Query Editor
Full syntax highlighting, field autocomplete, and history. Hit ⌘↵ to run against any stream.
Live Log Explorer
Virtualized rows, level filters, and a JSON inspector drawer for every event. Scroll 400k+ results without a hitch.
Interactive Histogram
See event volume at a glance and drag across the bars to narrow the time range — the query re-runs instantly.
Multiple Connections
Save every OpenObserve instance and org. Switch context from the titlebar — credentials stay on your machine.
Streams & Fields
Browse streams with sizes, filter fields inline, and one-click any field into your query. Metrics, Traces & Dashboards coming soon.
